# BAYAN Pro

BAYAN Pro is a modern WordPress block theme built for professional service businesses in Arabic and Gulf markets. It focuses on clarity, trust, bilingual presentation, and conversion-oriented service pages.

## Features
- Full Site Editing block theme
- True RTL support
- Arabic and English ready
- Reusable block patterns
- Service, case study, industry, testimonial, and team content types
- Global style variations
- Lightweight and responsive layout
- Conversion-focused service pages
- Request a Quote page structure
- Professional business demos

## Requirements
- WordPress 6.6+
- PHP 8.1+
- BAYAN Core plugin

## Installation
1. Upload and activate the `bayan-pro` theme.
2. Upload and activate the `bayan-core` plugin.
3. Import demo content if needed.
4. Go to Appearance > Editor to customize templates and patterns.
5. Set Homepage and Posts Page from Settings > Reading.
6. Configure company details from BAYAN Settings.
7. Build your navigation menu and assign it.
8. Add services, case studies, testimonials, industries, and team content.

## Theme Structure
- Templates for homepage, services, case studies, industries, FAQ, contact, and landing pages
- Reusable header/footer parts
- Multiple service and trust block patterns
- Style variations for light and dark brand directions

## Supported Content Types
- Services
- Case Studies
- Industries
- Testimonials
- Team Members

## Recommended Plugins
- Polylang or WPML
- Fluent Forms / WPForms / Contact Form 7
- Rank Math or Yoast SEO

## Localization
BAYAN Pro is built to support Arabic and English websites with proper RTL handling.

## Performance Notes
- Keep uploaded images optimized
- Avoid installing unnecessary animation plugins
- Use only required forms and integrations

## Support Scope
Theme support covers:
- Installation issues
- Demo import issues
- Theme bugs
- RTL layout issues

Theme support does not cover:
- Custom feature development
- Third-party plugin conflicts outside reasonable compatibility
- Custom coding requests
