# GrowthyTools Theme

قالب WordPress كلاسيكي RTL-first لمتجر **GrowthyTools**.

## التثبيت
1. انسخ المجلد `growthytools` إلى `wp-content/themes/`.
2. فعّل القالب من لوحة التحكم.
3. فعّل إضافة `growthytools-catalog` أولاً لضمان ظهور نوع المنشور `gt_template`.

## الإعداد
- من **المظهر > تخصيص** عدّل الألوان، نصوص الهيرو، أزرار CTA، مزايا المتجر، وشارات الثقة.
- أنشئ صفحة رئيسية وعيّن قالب الصفحة الرئيسية الافتراضي لـ WordPress.

## المزايا
- واجهة عربية أولاً مع RTL.
- صفحة رئيسية Conversion-focused.
- صفحة قالب فردي مع buy box لشراء مباشر بدون سلة.
- JSON-LD لصفحات القوالب.

## التوافق
- WordPress 6.4+
- PHP 8.0+
